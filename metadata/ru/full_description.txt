Особенности:
- текущая погода
- недельный прогноз (на 7 дней)
- 48-ми часовой прогноз
- карта погоды
- дождевой радар
- Предупреждения о погоде
- множество городов
- множество единиц измерения

Поддерживаемые языки:
- Английский
- Армянский
- Болгарский
- Босния
- Эсперанто
- Грузинский
- Испанский
- Китайский (упрощенный)
- Китайский (традиционный)
- Македония
- Немецкий
- Польский
- Португальский (БР)
- Русский
- Финский
- Французский
- голландский
- греческий
- Турецкий
- итальянский
- сербский
- украинский
- чешский

Данные погоды:
Погодные данные предоставляются <a href="https://openweathermap.org/">OpenWeatherMap.org</a>, под лицензией <a href="https://creativecommons.org/licenses/by-sa/4.0/">CC-BY-SA 4.0</a>.

Получить ключ API:
Перейдите на <a href="https://home.openweathermap.org/users/sign_up">OpenWeatherMap.org</a> и зарегистрируйтесь свободно. Активация ключа API может занять некоторое время!
